Key Features

  • Automatic navigation interception for internal links
  • Multiple pre-built transition types
  • Browser back/forward button support
  • Graceful fallback for unsupported browsers
  • Respects user's motion preferences
  • Responsive optimizations for mobile and desktop
  • Persistent header and footer during transitions
  • Custom transition types via data attributes

Usage Example

Simply add a data attribute to any link:

<a href="/page.html" data-transition-type="slide"> Navigate with Slide </a> <a href="/page.html" data-transition-type="fade"> Navigate with Fade </a> <!-- Exclude from transitions --> <a href="/page.html" class="no-transition"> Standard Navigation </a>
