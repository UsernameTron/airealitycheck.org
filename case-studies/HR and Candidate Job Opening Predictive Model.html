Project Overview

Tasked with analyzing customer experience challenges for TikTok Shop with partial data, I merged authenticated data with carefully synthesized estimates to create a comprehensive, strategic analysis. This framework addressed immediate gaps and laid the groundwork for future optimizations.

Innovative Approach & Creative Problem Solving

My approach integrated:

  • Data Authentication & Synthesis: Consolidated multiple verified data sources while intelligently estimating missing elements.
  • Predictive Analytics: Applied advanced ML techniques (Random Forest, SHAP) to forecast trends and derive actionable insights.
  • Dashboard Creation: Developed interactive dashboards that translate complex data into clear, executive-level insights.

This methodology underscores my ability to innovate under uncertainty and deliver strategic, actionable solutions.

Strategic Analysis & Actionable Insights
